# springboot-docker
Projeto básico para demonstrar a dockeirização de uma aplicação Spring Boot Java/Gradle

Basta clonar o projeto e utilizar a branch Master para rodar.

Artigo do qual esse projeto é referente: [Dockerizando uma aplicação Spring Boot/Gradle](https://www.linkedin.com/pulse/dockerizando-uma-aplica%25C3%25A7%25C3%25A3o-spring-bootgradle-felipe-neves)
